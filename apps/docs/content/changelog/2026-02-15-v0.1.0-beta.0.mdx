The first public beta of Keystone UI — a comprehensive React component library
built from the ground up for the Tailwind CSS v4 era.

<ChangeTag variant="added" />

- **58 components** spanning forms, layout, navigation, feedback, and overlays
- **Base UI** primitives wired up for accessibility and headless behavior
- **Tailwind CSS v4** theme with OKLCH color tokens and CSS-first configuration
- **CVA** (class-variance-authority) for component variants
- **Dark mode** support out of the box, driven by semantic tokens
- **Full TypeScript** support with type exports for every component
- **Subpath exports** so consumers tree-shake to a single component if they want
- **Documentation site** built with Fumadocs, including live demos and source viewers
