## Features

- Sticky-header table with 3-state column sort and column visibility toggle
- Row checkbox selection with indeterminate header state
- Inline cell editing — `Input` for the Subject, `DropdownMenu` for all enum cells (Status, Priority, Assignee, Category, Channel, Health)
- New-ticket `Modal` with required Subject validation and Status / Priority / Category / Channel / Assignee selectors
- Floating `SelectionBar` for bulk Change Status / Assign / Priority / Health / Channel / Export CSV / Delete (with `AlertDialog` confirmation)
- Drag-to-reorder rows when sort preset is "Manual" (`@dnd-kit/*`)
- Mobile toolbar collapses Sort / Status / Columns / View into a single `Drawer` + `Stepper` filters panel
