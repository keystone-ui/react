## Import

```tsx

```

## Variants

<ComponentPreview name="badge-variants" />

| Variant | Use Case |
| --- | --- |
| `default` | Primary emphasis badges |
| `secondary` | Subtle secondary labels |
| `destructive` | Error or danger indicators |
| `outline` | Bordered minimal badges |
| `ghost` | Low-emphasis hover-revealed badges |
| `link` | Underlined text-style badges |

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"default" \| "secondary" \| "destructive" \| "outline" \| "ghost" \| "link"` | `"default"` | Visual style variant |
| `size` | `"default" \| "sm"` | `"default"` | Size of the badge |
| `className` | `string` | - | Additional CSS classes |
