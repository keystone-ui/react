## Variants

The Alert supports semantic color variants for different message types:

<ComponentPreview name="alert-variants" />

| Variant | Use Case |
| --- | --- |
| `default` | General informational messages |
| `success` | Successful operations or confirmations |
| `warning` | Caution or attention needed |
| `error` | Errors or critical failures |
| `info` | Informational tips or notes |

## API Reference

### Alert

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"default" \| "success" \| "warning" \| "error" \| "info"` | `"default"` | Semantic color variant |
| `className` | `string` | - | Additional CSS classes |
