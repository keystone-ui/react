---
title: Alert
description: Prominent message banner for displaying important information, warnings, or errors to users.
---

## Import

```tsx
import { Alert, AlertTitle, AlertDescription } from "keystoneui/alert";
```

## Usage

```tsx
<Alert>
  <AlertTitle>Heads up!</AlertTitle>
  <AlertDescription>This is an informational alert message.</AlertDescription>
</Alert>
```
