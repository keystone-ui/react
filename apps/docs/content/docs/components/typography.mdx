title: Typography
description: Examples of utility classes for styling headings, paragraphs, lists, and other text elements.
---

Keystone UI does not ship any typography styles by default. This page shows how you can use Tailwind CSS utility classes to style your text.

## Demo

<ComponentPreview name="typography-default" />

## H1

```tsx
<h1 className="scroll-m-20 text-4xl font-extrabold tracking-tight text-balance">
  ...
</h1>
```
