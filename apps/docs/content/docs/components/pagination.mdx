## API Reference

### PaginationLink

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `href` | `string` | - | Page URL |
| `isActive` | `boolean` | `false` | Whether this is the current page |
| `className` | `string` | - | Additional CSS classes |
