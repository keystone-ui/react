---
title: InputGroup
description: Groups an input with adjacent buttons or icons for combined input actions.
---

## Import

```tsx
import { InputGroup, InputGroupInput, InputGroupButton } from "keystoneui/input-group";
```

## Usage

```tsx
<InputGroup>
  <InputGroupInput placeholder="Search..." />
  <InputGroupButton>Go</InputGroupButton>
</InputGroup>
```
