---
title: InputGroup
description: Groups an input with adjacent buttons, icons, or text add-ons for combined input actions.
---

## Import

```tsx
import {
  InputGroup,
  InputGroupInput,
  InputGroupButton,
  InputGroupAddon,
  InputGroupText,
} from "keystoneui/input-group";
```

## API Reference

### InputGroup

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | `"default" \| "sm"` | `"default"` | The size variant of the group |
| `className` | `string` | - | Additional CSS classes |
