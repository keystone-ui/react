title: Item
description: Generic list item primitive for building consistent list-based interfaces.

## Import

```tsx
import {
  Item,
  ItemMedia,
  ItemContent,
  ItemTitle,
  ItemDescription,
  ItemActions,
  ItemGroup,
  ItemSeparator,
} from "@keystoneui/react/item";
```
