---
title: NativeSelect
description: Native browser select dropdown for simple selection scenarios with optimal mobile support.
---

## Import

```tsx
import { NativeSelect } from "keystoneui/native-select";
```

## Usage

```tsx
<NativeSelect>
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
  <option value="3">Option 3</option>
</NativeSelect>
```
