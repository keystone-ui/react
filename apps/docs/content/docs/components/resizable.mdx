```tsx
import { ResizablePanelGroup, ResizablePanel, ResizableHandle } from "keystoneui/resizable";
```

## API Reference

### ResizablePanelGroup

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `direction` | `"horizontal" \| "vertical"` | - | The direction of the panel group |

### ResizablePanel

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `defaultSize` | `number` | - | Initial size as a percentage |
| `minSize` | `number` | - | Minimum size as a percentage |
| `maxSize` | `number` | - | Maximum size as a percentage |

### ResizableHandle

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `withHandle` | `boolean` | `false` | Show a visible drag handle indicator |
