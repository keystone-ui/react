---
title: Card
description: Structured container with header, content, and footer sections for grouping related information.
---

## Import

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter,
} from "@keystoneui/react/card";
```

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | `"md" \| "sm" \| "xs"` | `"md"` | The size variant of the card |
| `className` | `string` | - | Additional CSS classes |
