## Import

```tsx
import {
  Breadcrumb,
  BreadcrumbList,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbSeparator,
  BreadcrumbPage,
  BreadcrumbEllipsis,
} from "@keystoneui/react/breadcrumb";
```

## API Reference

### BreadcrumbLink

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `href` | `string` | - | The URL the breadcrumb link points to |
| `className` | `string` | - | Additional CSS classes |
