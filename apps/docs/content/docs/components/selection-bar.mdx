---
title: Selection Bar
description: Floating bulk-action toolbar that appears when items are selected in a list or table.
---

## Import

```tsx
import {
  SelectionBar,
  SelectionBarClose,
  SelectionBarGroup,
  SelectionBarLabel,
  SelectionBarBullet,
  SelectionBarLink,
  SelectionBarSeparator,
  SelectionBarButton,
} from "@keystoneui/react/selection-bar";
```

## API Reference

### SelectionBar

| Prop       | Type                       | Default  | Description                                                                |
| ---------- | -------------------------- | -------- | -------------------------------------------------------------------------- |
| `open`     | `boolean`                  | —        | Whether the bar is visible. Drives the 200ms enter/exit animation.         |
| `position` | `"fixed" \| "inline"`      | `"fixed"`| `fixed` anchors to the viewport bottom; `inline` lets the consumer place it. |

### SelectionBarButton

| Prop    | Type                                       | Default     | Description                                       |
| ------- | ------------------------------------------ | ----------- | ------------------------------------------------- |
| `tone`  | `"default" \| "success" \| "destructive"` | `"default"` | Semantic color treatment for hover and text.      |
| `shape` | `"text" \| "icon"`                         | `"text"`    | Use `icon` for square icon-only triggers.         |

Pass to a `DropdownMenuTrigger` via `render={<SelectionBarButton />}` to combine selection-bar styling with menu behavior.
