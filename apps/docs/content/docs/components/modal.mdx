## Import

```tsx
import {
  Modal,
  ModalTrigger,
  ModalContent,
  ModalHeader,
  ModalTitle,
  ModalDescription,
  ModalBody,
  ModalFooter,
  ModalClose,
} from "@keystoneui/react/modal";
```

## Sizes

The Modal supports multiple width presets:

<ComponentPreview name="modal-sizes" />

| Size | Width | Use Case |
| --- | --- | --- |
| `sm` | 400px | Simple confirmations |
| `default` | 500px | Standard dialogs |
| `lg` | 640px | Forms and detail views |
| `xl` | 780px | Complex content |
| `full` | Full screen | Immersive experiences |

## API Reference

### Modal

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `open` | `boolean` | - | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when open state changes |
| `defaultOpen` | `boolean` | `false` | Initial open state (uncontrolled) |

### ModalContent

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | `"sm" \| "default" \| "lg" \| "xl" \| "full"` | `"default"` | Modal width |
| `className` | `string` | - | Additional CSS classes |
