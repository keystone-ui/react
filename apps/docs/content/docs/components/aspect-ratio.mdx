title: AspectRatio

---

```tsx
import { AspectRatio } from "@keystoneui/react/aspect-ratio";
```

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `ratio` | `number` | - | The desired aspect ratio expressed as width / height (e.g. `16/9`) |
| `className` | `string` | - | Additional CSS classes |
