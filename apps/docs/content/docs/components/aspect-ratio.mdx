---
title: AspectRatio
description: Constrains content to a consistent width-to-height ratio for images, videos, or embeds.
---

## Import

```tsx
import { AspectRatio } from "keystoneui/aspect-ratio";
```

## Usage

```tsx
<AspectRatio ratio={16 / 9}>
  <img src="/image.jpg" alt="Landscape" />
</AspectRatio>
```
