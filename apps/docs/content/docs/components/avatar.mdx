## API Reference

### Avatar

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | `"default" \| "sm" \| "lg" \| "xl"` | `"default"` | Size of the avatar |
| `className` | `string` | - | Additional CSS classes |

### AvatarImage

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `src` | `string` | - | Image source URL |
| `alt` | `string` | - | Alt text for the image |
