---
title: Avatar
description: User avatar displaying a profile image with a text or icon fallback.
---

## Import

```tsx
import { Avatar, AvatarImage, AvatarFallback } from "keystoneui/avatar";
```

## Usage

```tsx
<Avatar>
  <AvatarImage src="/avatar.jpg" alt="User avatar" />
  <AvatarFallback>JD</AvatarFallback>
</Avatar>
```
