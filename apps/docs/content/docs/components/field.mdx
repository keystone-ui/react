## Import

```tsx

  Field,
  FieldLabel,
  FieldDescription,
  FieldError,
  FieldSet,
  FieldLegend,
  FieldGroup,
  FieldContent,
  FieldTitle,
  FieldSeparator,
} from "keystoneui/field";
```

## API Reference

### Field

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `orientation` | `"vertical" \| "horizontal"` | `"vertical"` | Layout direction of the field |
| `className` | `string` | - | Additional CSS classes |

### FieldLabel

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | - | Additional CSS classes |

### FieldError

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `show` | `boolean` | `true` | Whether to show the error message |
| `className` | `string` | - | Additional CSS classes |
