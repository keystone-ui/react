## API Reference

### toast()

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| `message` | `string` | - | The toast message |
| `description` | `string` | - | Optional description text |
| `action` | `ToastAction` | - | Action button configuration |
| `position` | `ToastPosition` | `"bottom-right"` | Position on screen |
| `duration` | `number` | `4000` | Duration in milliseconds |

### Toaster

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `position` | `ToastPosition` | `"bottom-right"` | Default position for all toasts |
| `richColors` | `boolean` | `false` | Use semantic colors for toast types |
