---
title: TagGroup
description: Managed group of tags with add and remove capabilities for multi-value selection.
---

## Import

```tsx
import { TagGroup, TagGroupItem } from "keystoneui/tag-group";
```

## Usage

```tsx
<TagGroup>
  <TagGroupItem>React</TagGroupItem>
  <TagGroupItem>TypeScript</TagGroupItem>
  <TagGroupItem>Tailwind</TagGroupItem>
</TagGroup>
```
