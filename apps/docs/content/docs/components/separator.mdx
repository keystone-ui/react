---
title: Separator
description: Visual divider for separating content sections horizontally or vertically.
---

## Import

```tsx
import { Separator } from "keystoneui/separator";
```

## Usage

<ComponentPreview name="separator-default" />

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | The orientation of the separator |
| `decorative` | `boolean` | `true` | Whether the separator is decorative (not announced by screen readers) |
| `className` | `string` | - | Additional CSS classes |
