## Secondary

Filled when unpressed (uses `bg-secondary`) and inverted to `bg-foreground` when pressed. Useful when an unpressed toggle would otherwise read as empty space — e.g., a 3-pill segmented control where only one is selected.

## API Reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"default" \| "outline" \| "secondary"` | `"default"` | Visual style variant |
| `size` | `"default" \| "sm" \| "lg"` | `"default"` | Size of the toggle |
| `pressed` | `boolean` | - | Controlled pressed state |
| `onPressedChange` | `(pressed: boolean) => void` | - | Callback when pressed state changes |
| `disabled` | `boolean` | `false` | Whether the toggle is disabled |
