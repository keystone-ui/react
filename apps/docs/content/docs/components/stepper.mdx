---
title: Stepper
description: Multi-step progress indicator for guiding users through sequential workflows.
---

## Import

```tsx
import { Stepper, StepperItem } from "keystoneui/stepper";
```

## Usage

```tsx
<Stepper activeStep={1}>
  <StepperItem>Account</StepperItem>
  <StepperItem>Profile</StepperItem>
  <StepperItem>Review</StepperItem>
</Stepper>
```
