## API Reference

### Stepper

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `activeStep` | `number` | `0` | The currently active step index |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Layout direction |

### StepperStep

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | - | Additional CSS classes |
