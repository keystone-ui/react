## API Reference

### Label

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `htmlFor` | `string` | - | ID of the associated form control |
| `className` | `string` | - | Additional CSS classes |

### Description

Renders muted helper text below a form control.

### ErrorMessage

Renders destructive-colored error text below a form control.
