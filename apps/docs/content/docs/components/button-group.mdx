## API Reference

### ButtonGroup

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Layout direction |
| `size` | `"default" \| "sm" \| "lg"` | `"default"` | Size of buttons in the group |
| `className` | `string` | - | Additional CSS classes |
