---
title: ButtonGroup
description: Groups related buttons together with consistent spacing and connected styling.
---

## Import

```tsx
import { ButtonGroup } from "keystoneui/button-group";
```

## Usage

```tsx
<ButtonGroup>
  <button>Left</button>
  <button>Center</button>
  <button>Right</button>
</ButtonGroup>
```
