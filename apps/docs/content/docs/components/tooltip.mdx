## API Reference

### TooltipContent

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"top"` | Preferred placement side |
| `sideOffset` | `number` | `4` | Offset from the trigger in pixels |
| `className` | `string` | - | Additional CSS classes |
