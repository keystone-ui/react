---
title: Tooltip
description: Small floating label that appears on hover to provide additional descriptive text.
---

## Import

```tsx
import { Tooltip, TooltipTrigger, TooltipContent } from "keystoneui/tooltip";
```

## Usage

```tsx
<Tooltip>
  <TooltipTrigger>Hover me</TooltipTrigger>
  <TooltipContent>Helpful tooltip text</TooltipContent>
</Tooltip>
```
