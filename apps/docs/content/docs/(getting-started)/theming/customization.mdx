Keystone UI's theme is built entirely on CSS custom properties. You own the theme CSS in your project and can customize every token.

## Complete Theme Reference

Below is the full theme CSS. Copy this into your `globals.css` after `@import "@keystoneui/react/base.css"` and modify any values.

<Callout type="info">
  You can also generate this automatically via the [Registry CLI](/docs/installation/registry): `npx shadcn add https://keystoneui.io/r/default.json`
</Callout>

  Always define both `:root` (light) and `.dark` (dark) values to maintain

</Callout>

## Adding New Colors

Define variables in `:root` / `.dark`, then register them in the `@theme inline` block:

```css title="globals.css"
:root {
  --warning: oklch(0.84 0.16 84);
  --warning-foreground: oklch(0.28 0.07 46);
}

.dark {
  --warning: oklch(0.41 0.11 46);
  --warning-foreground: oklch(0.99 0.02 95);
}

@theme inline {
  --color-warning: var(--warning);
  --color-warning-foreground: var(--warning-foreground);
}
```

Use `bg-warning text-warning-foreground` in your components.

## Using OKLCH

We recommend using [oklch.com](https://oklch.com/) to pick colors. OKLCH provides:

- **Perceptual uniformity** -- Equal changes in L (lightness) look equally different
- **Wide gamut** -- Access to colors outside sRGB on modern displays
- **Intuitive** -- L (lightness 0-1), C (chroma 0-0.4), H (hue 0-360)

```css
/* OKLCH format: oklch(lightness chroma hue) */
--primary: oklch(0.55 0.2 250); /* Medium blue */
--primary: oklch(0.55 0.25 150); /* Vivid green */
--primary: oklch(0.55 0.2 30); /* Warm orange */
```

## Fonts

Keystone UI uses the `--font-sans` CSS variable for the base font family. You need to load the font and define this variable in your theme.

### Next.js (recommended)

Use `next/font/google` for automatic self-hosting with zero external requests:

```tsx title="app/layout.tsx"
import { Inter } from "next/font/google";

const font = Inter({ subsets: ["latin"], variable: "--font-inter" });

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en" className={font.variable}>
      <body>{children}</body>
    </html>
  );
}
```

Then set `--font-sans` in your `globals.css`:

```css title="globals.css"
:root {
  --font-sans: var(--font-inter);
}
```

### Other frameworks

Add a Google Fonts link to your `<head>` and set the font family directly:

```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@100..900&display=swap" rel="stylesheet">
```

```css title="globals.css"
:root {
  --font-sans: "Inter", sans-serif;
}
```
