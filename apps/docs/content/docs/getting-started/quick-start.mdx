## Quick Install

### 1. Install Package

```bash tab="pnpm"
pnpm add @keystoneui/react
```

```bash tab="npm"
npm install @keystoneui/react
```

```bash tab="yarn"
yarn add @keystoneui/react
```

### 2. Set Up CSS

Run the registry command to generate the complete theme in your `globals.css`:

```bash
npx shadcn add https://keystoneui.io/r/keystone-style.json
```

This adds the base CSS import, dark mode variant, Tailwind color registration, default tokens, and base layer -- everything you need.

### 3. Add Source Directive

Add the `@source` directive to your `globals.css` so Tailwind can find component classes:

```css title="globals.css"
@import "tailwindcss";
@import "@keystoneui/react/base.css";

@source "../node_modules/@keystoneui/react/src/**/*.{ts,tsx}";

/* ... rest of generated theme ... */
```

<Callout type="info">
  Without the `@source` directive, Tailwind CSS v4 won't scan Keystone UI's
  source files and component styles will be missing.
</Callout>

### 4. Use Components

```tsx
import { Button } from "@keystoneui/react/button";
import { Input } from "@keystoneui/react/input";
import {
  Card,
  CardContent,
  CardDescription,
  CardHeader,
  CardTitle,
} from "@keystoneui/react/card";

export default function LoginCard() {
  return (
    <Card className="w-[380px]">
      <CardHeader>
        <CardTitle>Sign In</CardTitle>
        <CardDescription>Enter your credentials to continue.</CardDescription>
      </CardHeader>
      <CardContent className="space-y-4">
        <Input type="email" placeholder="Email" />
        <Input type="password" placeholder="Password" />
        <Button className="w-full">Sign In</Button>
      </CardContent>
    </Card>
  );
}
```

If you prefer full control over the theme or aren't using the shadcn CLI, set up the CSS manually.

### 1. Install Package

```bash tab="pnpm"

```

```bash tab="npm"
npm install @keystoneui/react
```

```bash tab="yarn"

```

## Import Pattern

Every component uses subpath exports. Import only what you need:

```tsx
import { Button } from "@keystoneui/react/button";
import { Badge } from "@keystoneui/react/badge";
import { Tabs, TabsList, TabsTrigger, TabsContent } from "@keystoneui/react/tabs";
```

<Callout type="warn">
  There is no barrel export (`@keystoneui/react`). Always use the subpath import
  pattern shown above.
</Callout>

## Dark Mode

Keystone UI supports dark mode out of the box. Add the `dark` class to your `<html>` element:

```html
<html class="dark">
  ...
</html>
```

All components will automatically adapt to dark mode with carefully tuned colors.
