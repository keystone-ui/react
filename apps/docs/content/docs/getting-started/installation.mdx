This page covers the full installation with TypeScript configuration and verification. For a faster setup, see the [Quick Start](/docs/getting-started/quick-start).

## Install Package

```bash tab="pnpm"
pnpm add @keystoneui/react
```

```bash tab="npm"
npm install @keystoneui/react
```

```bash tab="yarn"
yarn add @keystoneui/react
```

## Set Up CSS

### Via Registry (recommended)

The CLI generates the complete theme in your `globals.css`:

```bash
npx shadcn add https://keystoneui.io/r/keystone-style.json
```

This adds the base CSS import, dark mode variant, Tailwind color registration, default tokens, and base layer.

Then add the `@source` directive so Tailwind can find component classes:

```css title="globals.css"
@import "tailwindcss";
@import "@keystoneui/react/base.css";

@source "../node_modules/@keystoneui/react/src/**/*.{ts,tsx}";

/* ... rest of generated theme ... */
```

See the [Registry guide](/docs/getting-started/registry) for namespace config, swapping themes, and creating your own.

### Manual Setup

If you prefer not to use the CLI, see the complete CSS block in the [Quick Start manual setup](/docs/getting-started/quick-start#manual-setup) section. Copy it into your `globals.css` and customize the token values as needed.

<Callout type="info">
  The `@source` directive tells Tailwind CSS v4 where to find classes used by
  Keystone UI components. Without it, component styles may be missing.
</Callout>

## TypeScript Configuration

If you're using TypeScript, add path aliases for better IDE support:

```json title="tsconfig.json"
{
  "compilerOptions": {
    "paths": {
      "@keystoneui/react/*": ["./node_modules/@keystoneui/react/src/*"]
    }
  }
}
```

## Verify Installation

Create a simple component to verify everything is working:

```tsx title="app/page.tsx"
import { Button } from "@keystoneui/react/button";

export default function Page() {
  return (
    <div className="flex min-h-screen items-center justify-center">
      <Button>Hello Keystone UI</Button>
    </div>
  );
}
```

If you see a styled button, you're all set!
